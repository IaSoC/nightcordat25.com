## Overview

This project is an **unofficial recreation** of the countdown page that appeared in *Colorful Stage! The Movie: A Miku Who Can't Sing. *

In the movie, this page was created by **Akiyama Mizuki**, shown briefly on Kanade’s computer desktop when she opens a can of energy drink.

When I noticed this detail during my second watch, I realized that **SEGA hadn’t registered** `nightcordat25.com`.

So—I bought it, and decided to rebuild that page myself.

**With Assistense from LLM. I will explain later.**

## About the Page

This website **is not** an official production.

I’m not Mizuki, nor a SEGA staff member.

I simply understand a bit of programming — mainly C and Python, not front-end —

but I wanted to recreate that glowing midnight countdown.

**That's why I use LLM for coding. I don't have skills on frot-end. Still, I can recognise the funcions of css sytles. xD**

The fonts weren’t easy to match.

The digits were redrawn by hand with a pen tool and imported into FontForge.

The Japanese text was too blurry to identify, so I used common replacements.

This, I suppose, is the limit of my “faithful recreation.”
